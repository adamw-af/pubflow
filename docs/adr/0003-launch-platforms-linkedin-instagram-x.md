# Launch with LinkedIn, Instagram, and X only

v1 supports three platforms: LinkedIn, Instagram, and X (Twitter). All nine planned platforms (+ Facebook, TikTok, YouTube, Bluesky, Threads, Pinterest) are deferred to post-launch increments. Each platform requires its own OAuth app registration, API approval process, and publishing logic — shipping all nine at once would significantly delay launch. LinkedIn has the most developer-friendly API; Instagram requires Meta app review (~1-2 weeks); X requires a paid developer account (~$100/month, paid by PubFlow, not per user).
