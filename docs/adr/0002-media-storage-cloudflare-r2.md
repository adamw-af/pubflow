# Media storage via Cloudflare R2 with presigned uploads

Media files are stored in Cloudflare R2. The browser uploads directly to R2 using a presigned URL generated by a Convex action (via the R2 S3-compatible API) — the app server never proxies the bytes. Convex's built-in file storage was rejected because its 1 GB free tier is not viable for a video-heavy scheduling tool. R2 has no egress fees and scales cheaply.
